# Repict
Image editing function library

## Author
Nathaniel Schutte

## Help
Commandline tool only as of right now.
### Build:
```
make
```
### How to use:
```
repict <image.bmp> -f <function ID> // default write to out.bmp
repict <image.bmp> -f <function ID> -o <edited_image.bmp>
```
